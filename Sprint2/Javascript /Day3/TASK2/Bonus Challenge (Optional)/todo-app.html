LocalStorage & DOM Manipulation Q&A

1. What is localStorage and how does it differ from sessionStorage?
localStorage is a web storage API that allows you to store key-value pairs in a web browser with no expiration time. Unlike sessionStorage, which clears when the page session ends (when the tab is closed), localStorage persists even when the browser is closed and reopened. In this app, we use localStorage to save tasks so they remain available even after the page is refreshed or the browser is closed.
2. How do you add an item to localStorage?
You can add an item to localStorage using the setItem() method: 
localStorage.setItem('key', 'value');
Note that localStorage can only store strings, so when storing objects or arrays, you need to use JSON.stringify() to convert them to a string first.
3. How do you retrieve data from localStorage?
You can retrieve data using the getItem() method: 
const data = localStorage.getItem('key');
If you stored an object or array, you'll need to parse it back using JSON.parse(): 
const tasks = JSON.parse(localStorage.getItem('tasks')) || [];
4. What is the purpose of the DOMContentLoaded event?
The DOMContentLoaded event fires when the initial HTML document has been completely loaded and parsed, without waiting for stylesheets, images, and subframes to finish loading. We use it to ensure our JavaScript code runs only after the DOM is ready to be manipulated: 
document.addEventListener('DOMContentLoaded', function() {
    // Your code here
});
5. How does this app dynamically update the task list?
The app uses several DOM manipulation techniques:
  • Adding tasks: When a new task is added, we create new DOM elements (li, input, span, button) and append them to the task list.
  • Updating tasks: Event listeners are added to the checkboxes to toggle the 'completed' class and update the task's status in localStorage.
  • Deleting tasks: Clicking the delete button removes the task from both the DOM and localStorage.
  • Loading tasks: On page load, we retrieve tasks from localStorage and dynamically generate the task list.
